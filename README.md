# Biblioteca App

Aplicación de gestión de biblioteca desarrollada en C# con Avalonia UI.
Trabajo para el módulo de Programación - DAM.

## Características

- Gestión de libros y audiolibros
- Préstamos de libros con control de disponibilidad
- Sistema de valoraciones con puntuación media automática
- Exportación e importación de catálogo en CSV
- Base de datos SQLite local
- Patrón MVC

## Tecnologías

- C# / .NET 9
- Avalonia UI 11 (interfaz multiplataforma)
- SQLite (base de datos local)

## Cómo ejecutar

```bash
dotnet restore
dotnet run
```
